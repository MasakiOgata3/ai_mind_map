# 欲しい機能
 
既存のマインドマップのノードの内容を踏まえ、
指定したノードから関連性の高いノードを新たにAIが追加する機能
 
# 処理の流れ
 
1. 特定のノードの AI ボタンを押す
2. Gemini API は既存のノードの内容をすべて読み込む
3. ボタンを押されたノードから派生するアイデアを新規ノードとして追加する
 
# 注意点
 
- 読み込み中はローディングの表示を出す
- 新規ノードの追加はリロードしなくても反映されるようにする
- AI が作成したアイデアは識別できるようにする
 
# Gemini API の使用方法
 
`.env.local` というファイルに API キーを書いています。
そこに `GEMINI_API_KEY=xxxxxxx` という形で書いてあるので、その前提で進めてください。
`.env.local` ファイル自体は読み込まないでください。
 
Gemini API の利用にあたっては最新のドキュメントを確認し、無料、かつ高速で使えるモデルを選ぶこと
 
また、クライアント側で API キーを読み込むとセキュリティ上のリスクがあるため、必ず Next.js の API ルートで処理を行うこと。
（`NEXT_PUBLIC_` プレフィックスの使用は絶対に避けること）
