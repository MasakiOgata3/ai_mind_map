# 作りたいアプリのイメージ
 
Next.js で作るマインドマップアプリ「AI Mind Map」
 
# 欲しい機能
 
- 中心の要素から派生した要素を伸ばしながらアイデアを広げられる
- 各要素は追加・編集・削除できる
- ドラッグ＆ドロップでの配置換え機能はなく、ちょうどいい場所に自動配置される
- 親の要素を削除したら、子の要素もすべて削除される
- 各要素の色を変えられる
 
# 制約事項
 
## 技術制約
 
- **Next.js** を使用
- **Tailwind CSS 4.x** を使用（最新設定方法準拠）
- **shadcn/ui** コンポーネントを積極活用
- **TypeScript** による型安全な実装
- **Jest + React Testing Library** でのテスト実装
- **機能要件に必要なライブラリ** を活用する
 
## デザイン制約
- **ライトモードのみ対応**（ダークモード非対応）
- **Apple Human Interface Guidelines** 準拠
- **Core Web Vitals** 最適化必須
- **アクセシビリティ** 考慮（コントラスト比・操作性）
 
## インフラ制約
 
- **データベース使用禁止**（localStorage活用）
- **指定した外部API以外を使用しない**
- **静的ホスティング対応**（Vercel等）
